# Юридические документы Precettore — ЧЕРНОВИКИ

Пакет обязательных документов под 152-ФЗ для платформы Precettore (precettore.ru).

> ⚠️ **Статус: черновики.** Это рабочие заготовки, подготовленные под реальную
> модель данных платформы. Перед публикацией их обязан проверить профильный юрист —
> особенно из-за обработки данных **несовершеннолетних** (школьники 7–11 классов).
> Это не заменяет юридическую консультацию.

## Плейсхолдеры — заполнить перед публикацией

Везде в тексте помечено `【…】`. Их нельзя опубликовать «как есть»:

- `【ОПЕРАТОР】` — кто оператор ПДн: ФИО (самозанятый) / ИП ФИО / ООО «…»
- `【ИНН】` — ИНН оператора
- `【АДРЕС】` — адрес для корреспонденции
- `【EMAIL】` — контактный email для запросов субъектов ПДн
- `【ДАТА】` — дата утверждения редакции
- `【РЕГИОН_БД】` — где физически находится БД (после миграции на РФ-хостинг — 242-ФЗ)

## Прежде чем публиковать — сделать (см. аудит)

1. Оформить статус оператора (самозанятый / ИП) — без него нельзя указать `【ОПЕРАТОР】`.
2. Мигрировать БД на российский хостинг (242-ФЗ) — иначе Политика будет декларировать
   то, чего нет.
3. Подать уведомление в Роскомнадзор (обработка + трансграничная передача) — `pd.rkn.gov.ru`.
4. Включить RLS и закрыть Storage-бакеты — иначе «меры защиты» (ст. 19) в Политике
   не соответствуют действительности.
